# streamdesk_profiles

Stream Deck profiles for DCS Flight Panels:

- Mi-8MTV2
- Ka-50 III
- OH-58D Kiowa Warrior

[![Demo Video](images/kdpv.png)](https://youtu.be/sFP8s7s-6wk)

## Installation and usage

1. Install DCS Flight Panels:
https://github.com/DCS-Skunkworks/DCSFlightpanels

2. Install DCS-BIOS:
https://github.com/DCS-Skunkworks/dcs-bios

3. Open the profile you need in dcsfp.exe.

4. Start DCS.

5. Enjoy.
